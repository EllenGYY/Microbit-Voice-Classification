Create a software representation of the micro:bit's 5x5 LED matrix. The display is initially blank.

Parameters
mapThe mapping information that relates pin inputs/outputs to physical screen coordinates.
idThe id the display should use when sending events on the MessageBus. Defaults to DEVICE_ID_DISPLAY.
