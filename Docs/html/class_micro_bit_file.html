int MicroBitFile::close ( )

Closes this MicroBitFile instance

Returns
MICROBIT_OK on success, MICROBIT_NOT_SUPPORTED if the file handle is invalid.
Note
MicroBitFiles are opened at construction and are implicitly closed at destruction. They can be closed explicitly using this member function.
