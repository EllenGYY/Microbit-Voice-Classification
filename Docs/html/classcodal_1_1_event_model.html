{
//do something
}

inline

Register a listener function.

An EventModel implementing this interface may optionally choose to override this method, if that EventModel supports asynchronous callbacks to user code, but there is no requirement to do so.

Parameters
componentThe source component of messages to listen for.
valueThe value of messages to listen for. Events with any other values will be filtered. Use DEVICE_EVT_ANY to receive events of any value.
handlerThe function to call when an event is received.
flagsUser specified, implementation specific flags, that allow behaviour of this events listener to be tuned.
Returns
DEVICE_OK on success, or any valid error code defined in "ErrNo.h". The default implementation simply returns DEVICE_NOT_SUPPORTED.
void onButtonBClicked(Event)
{
//do something
}
// call onButtonBClicked when ever a click event from buttonB is detected.
uBit.messageBus.listen(device.buttonB, DEVICE_BUTTON_EVT_CLICK, onButtonBClick);
