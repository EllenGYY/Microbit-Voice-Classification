Detailed Description

Class definition for a BitmapFont This class represents a font that can be used by the display to render text.

A BitmapFont is typically 5x5, but up to 8x8. Each Row is represented by a byte in the array.

Row Format:

| Bit 7 | Bit 6 | Bit 5 | Bit 4 | Bit 3 | Bit 2 | Bit 1 | Bit 0 |

| N/A | N/A | N/A | Col 1 | Col 2 | Col 3 | Col 4 | Col 5 | | 0x80 | 0x40 | 0x20 | 0x10 | 0x08 | 0x04 | 0x02 | 0x01 |

Example: { 0x08, 0x08, 0x08, 0x0, 0x08 }

The above will produce an exclaimation mark on the second column in from the left.

We could compress further, but the complexity of decode would likely outweigh the gains.
